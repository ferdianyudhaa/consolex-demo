Deskripsi Aplikasi

Aplikasi ini adalah mockup untuk mensimulasikan peluruhan radioaktif dari berbagai unsur yang sering digunakan dalam Fisika Medis dan Fisika Inti. Pengguna dapat memilih jenis isotop, memasukkan aktivitas awal, dan menentukan durasi simulasi. Melalui tiga tombol kontrol, aplikasi akan mengekstrak data, menghitung prediksi unsur turunan (daughter isotope), dan memvisualisasikan grafik kurva peluruhannya secara real-time di atas canvas.

Konsep Numerik & Algoritma

Aplikasi ini mengimplementasikan dua pendekatan matematis. Untuk peluruhan tunggal, digunakan persamaan eksponensial standar: $$N(t) = N_0 \cdot e^{-\lambda t}$$. Namun, untuk unsur dengan peluruhan berantai (Successive Decay) seperti Mo-99 menjadi Tc-99m, algoritma menggunakan Persamaan Bateman untuk menghitung aktivitas unsur anak: $$N_2(t) = \frac{\lambda_1}{\lambda_2 - \lambda_1} N_0 (e^{-\lambda_1 t} - e^{-\lambda_2 t})$$. Pada pengembangan fungsional selanjutnya, kalkulasi kontinu ini akan didiskritisasi menggunakan Metode Euler ($$N_{i+1} = N_i - \lambda N_i \Delta t$$) untuk mengakomodasi variabel laju eliminasi biologis pasien secara dinamis.
